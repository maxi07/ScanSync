To connect to OneDrive, you will need to create your own app registration on Microsoft Azure. This process is free of charge and only takes a few steps. After you followed the process, entery our app details here.

Sign Into your Azure Account

Head to portal.azure.com and sign up for a free account.

Head to App Registrations

This can also be found by using the search bar.

Create a new app registration

Enter details for registration

Give it any name, select Personal Microsoft accounts only.
Select Web as Redirect URI and enter http://:5001/getAToken. Double check capitalization.

Get Application ID

Take note of the Application (client) ID.

Create client secret

Get to Certificates & secrets, click New client secret, give it a name and set expiry to maximum.

Copy client secret

Take note of client secret. Now fill the two copied values into the next step.
